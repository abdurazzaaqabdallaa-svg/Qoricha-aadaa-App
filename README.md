
# 🌿 Qoricha-aadaa-App

Appilikeeshinii weebii (PWA) beekumsa qorichoota aadaa Oromoo walitti qabuu, mallattoolee dhibee addaan baasuu fi karaa teeknoolojii ammayyaatiin qajeelfama itti fayyadamaa kennuuf uumame.

### ✨ Akkaataa itti hojjetu (Features)
* **Barbaacha Salphaa (Live Search):** Maqaa qorichaa ykn faayidaa isaa galchuun kalliattiin odeeffannoo argachuu.
* **Offline Access:** Service Worker fayyadamuun internet malee iddoo kamittuu ni hojjeta.
* **Mobile Ready:** "Add to Home Screen" fayyadamanii bilbila irratti akka Appii fe'uun ni danda'ama.
* **Modern Interface:** Dizaayiniin isaa salphaa fi ifa kan ta'e (User-Friendly UI).

### 🛠️ Meeshaalee Fayyadame (Tech Stack)
* **HTML5 & CSS3:** Ijaarsa bu'uuraa fi bifa miidhagaa (Modern UI) uumuuf.
* **JavaScript (Vanilla):** Hojii barbaachaa (Search logic) fi hojii Appichaa ittiin hoogganuuf.
* **Web Manifest & Service Worker:** Appicha PWA gochuu fi Offline hojjachiisuuf.
* **GitHub Pages:** Appicha hundaaf bilisaan "Host" gochuuf.

### 🎯 Kaayyoo Pirojektichaa (Project Vision)
Beekumsa qoricha aadaa keenyaa kan qancaruu irraa oolchuun gara dijitaalaatti jijjiiruu fi dhaloota dhufuuf salphaatti akka dhaqqabamu gochuudha.

### ⚠️ Akeekkachiisa (Disclaimer)
Odeeffannoon as keessatti dhiyaate beekumsa aadaa qofa irratti kan hunda'eedha. Osoo hin fayyadamin dura ogeessa qoricha aadaa ykn ogeessa fayyaa mariisissuun gorfamaa dha.
