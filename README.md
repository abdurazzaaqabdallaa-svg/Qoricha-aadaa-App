# 🌿 Qoricha-aadaa-App (Xibb-un-Nabawi)

Appilikeeshinii weebii (PWA) beekumsa qorichoota aadaa fi qajeelfama fayyaa kitaaba **Xibb-un-Nabawi** irratti hundaa'an walitti qabu, akkasumas karaa teeknoolojii ammayyaatiin dhaloota haaraaf qajeelfama itti fayyadamaa kennuuf uumame.

### ✨ Akkaataa Itti Hojjetu (Features)
* **Barbaacha Salphaa (Live Search):** Maqaa qorichaa (Haba-suudaa, Rummaana, Jinjiibila...) ykn faayidaa isaa galchuun kalliattiin odeeffannoo argachuu.
* **Maddoota 32n Guutuu:** Qorichoota, eegumsa fayyaa dubartootaa, daa'immanii, dargaggootaa, maanguddootaa fi qajeelfama jireenyaa guutuu hamwateera.
* **Offline Access:** Service Worker fayyadamuun internet malee iddoo kamittuu (Offline) ni hojjeta.
* **Mobile Ready:** "Add to Home Screen" fayyadamanii bilbila irratti akka Appii fe'uun ni danda'ama.
* **Modern Interface:** Dizaayiniin isaa salphaa fi ifa kan ta'e (User-Friendly UI).

### 🛠️ Meeshaalee Fayyadame (Tech Stack)
* **HTML5 & CSS3:** Ijaarsa bu'uuraa fi bifa miidhagaa (Modern UI) uumuuf.
* **JavaScript (Vanilla):** Hojii barbaachaa (Search logic) fi kaardii qorichootaa 32n saffisaan dhalataa gochuuf.
* **Web Manifest & Service Worker:** Appicha PWA gochuu fi Offline hojjachiisuuf.
* **GitHub Pages:** Appicha hundaaf bilisaan "Host" gochuuf.

### 🎯 Kaayyoo Pirojektichaa (Project Vision)
Beekumsa qoricha aadaa fi qoricha bifa xinnabaawii (Xibb-un-Nabawi) kan qancaruu irraa oolchuun gara dijitaalaatti jijjiiruu fi dhaloota dhufuuf salphaatti akka dhaqqabamu gochuudha.

### ⚠️ Akeekkachiisa (Disclaimer)
Odeeffannoon as keessatti dhiyaate beekumsa aadaa fi xi-fayyaa irratti kan hunda'eedha. Osoo hin fayyadamin dura ogeessa qoricha aadaa ykn ogeessa fayyaa mariisissuun gorfamaa dha.
