# 🌿 Qoricha-aadaa-App

Appilikeeshinii weebii beekumsa qorichoota aadaa walitti qabuu, mallattoolee dhibee addaan baasuu fi karaa teeknoolojii ammayyaatiin qajeelfama itti fayyadamaa kennuuf uumame.

### ✨ Akkaataa itti hojjetu (Features)
* **Barbaacha Salphaa (Quick Search):** Maqaa qorichaa ykn mallattoo dhibee galchuun odeeffannoo argachuu.
* **Qajeelfama Fayyadamaa (Usage Guide):** Akkaataa qophiitti fi itti fayyadama isaa gabaasa qulqulluu ta'een agarsiisa.
* **Modern Database:** Odeeffannoo qorichaa bifa koodiitiin (JavaScript) gurmeessuu.

### 🛠️ Meeshaalee Fayyadame (Tech Stack)
* **HTML5 & CSS3:** Bifa bareedaa fi nama hawwatu uumuuf.
* **JavaScript:** Hojii barbaachaa (Search logic) fi hojii appichaa (Functionality) hojjechuuf.
* **GitHub Pages:** Appicha hundaaf akka dhiyaatu (Live hosting) gochuuf.

### 🎯 Kaayyoo Pirojektichaa (Project Vision)
Beekumsa qoricha aadaa keenyaa du'a irraa oolchuun gara dijitaalaatti jijjiiruu fi dhaloota dhufuuf salphaatti akka ghaqqabamu gochuudha.
